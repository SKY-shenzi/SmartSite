# 材料区传感器历史数据表（金仓数据库）

本模块按课程要求使用国产数据库金仓 KingbaseES。金仓只负责材料监控区的数据沉淀与趋势分析；签到打卡继续使用云数据库，方便不同手机、不同端之间共享打卡任务和打卡记录。

```text
华为云 IoTDA -> HarmonyOS App -> Kingbase Material API -> 金仓数据库 -> 材料历史折线图
```

## 建表 SQL

在金仓数据库中执行：

```sql
create table if not exists material_sensor_records (
  id bigint generated by default as identity primary key,
  zone_code varchar(50) not null,
  zone_name varchar(100) not null,
  device_id varchar(200) not null,
  temp numeric(10,2),
  humi numeric(10,2),
  dist numeric(10,2),
  lumi numeric(10,2),
  lamp_status varchar(50),
  shadow_version varchar(50),
  created_at timestamp default current_timestamp
);

create index if not exists idx_material_sensor_records_zone_time
on material_sensor_records(zone_code, created_at desc);

create table if not exists material_zones (
  id bigint generated by default as identity primary key,
  zone_code varchar(50) not null unique,
  zone_name varchar(100) not null,
  material_type varchar(50) not null,
  manager_name varchar(50),
  manager_phone varchar(30),
  temp_min numeric(10,2),
  temp_max numeric(10,2),
  humi_min numeric(10,2),
  humi_max numeric(10,2),
  fan_auto_enabled boolean default true,
  alarm_enabled boolean default true,
  created_at timestamp default current_timestamp
);
```

## 当前字段含义

| 字段 | 说明 |
| --- | --- |
| zone_code | 材料区编码，例如 cement_zone |
| zone_name | 材料区名称，例如水泥存储区 |
| device_id | 华为云 IoTDA 设备 ID |
| temp | 温度 Temp |
| humi | 湿度 Humi |
| dist | 距离 Dist |
| lumi | 光照 Lumi |
| lamp_status | 灯开关状态 LampST |
| shadow_version | 华为云设备影子版本 |
| created_at | 入库时间 |

## 当前金仓业务表

| 表名 | 用途 |
| --- | --- |
| users | 材料管理员、材料区负责人，不用于签到 |
| material_zones | 水泥区、木材区等材料区配置 |
| material_sensor_records | 华为云 IoTDA 材料传感器历史数据 |
| material_alert_rules | 温度、湿度等阈值告警规则 |
| material_alert_events | 超温、超湿等告警事件记录 |
| digital_twin_assets | 数字孪生场景里的区域、风扇、传感器等对象 |

## 数字孪生说明

本项目采用轻量级 2D 数字孪生方案。材料区真实设备通过华为云 IoTDA 上报温度、湿度、光照等数据，App 将数据写入金仓数据库。管理员进入“材料区数字孪生”页面后，系统从金仓读取材料区配置、最新传感器数据、告警事件和孪生对象，将真实材料区映射成软件中的虚拟材料区。

当前孪生联动规则：

| 真实对象 | 孪生对象 | 联动效果 |
| --- | --- | --- |
| 水泥存储区 | 水泥存储区孪生体 | 温湿度正常显示绿色，超限显示橙色或红色 |
| 温湿度传感器 | 传感器孪生对象 | 有历史数据时显示在线，暂无数据时显示等待数据 |
| 自动风扇 | 风扇孪生对象 | 温度超过阈值时显示自动开启 |
| 告警规则 | 告警事件列表 | 超温、超湿后写入 material_alert_events 并在 App 展示 |

## 后端接口

接口代码在：

```text
D:\SmartSiteServices\kingbase-material-api
```

启动后 App 会调用：

```text
POST http://电脑IP:8088/api/material-records
GET  http://电脑IP:8088/api/material-records?zoneCode=cement_zone&limit=24
GET  http://电脑IP:8088/api/material-zones
GET  http://电脑IP:8088/api/material-alerts?zoneCode=cement_zone&limit=24
GET  http://电脑IP:8088/api/twin-assets
```

## 数据库分工

| 模块 | 数据库 | 原因 |
| --- | --- | --- |
| 签到打卡 | 云数据库 Supabase | 多台手机和管理员端可以共享任务与记录 |
| 材料监控历史 | 本地金仓 KingbaseES | 满足国产数据库课程要求，并保存温湿度等历史数据用于折线图 |
| 视频监控 | 不入库 | 当前使用实时流地址，不需要保存视频数据 |

## App 配置

App 端配置在：

```text
entry/src/main/ets/common/MaterialDatabaseConfig.ets
```
