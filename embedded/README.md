# 嵌入式终端子系统

本目录包含智慧工地系统各区域的嵌入式终端设备程序，基于海思 WS63 (Hi3861V100) 平台开发，通过 WiFi + MQTT 连接华为云 IoTDA 平台。

## 目录

| 子项目 | 路径 | 说明 |
|--------|------|------|
| 木材区监控 | [timber-area/](timber-area/) | 木材堆放区环境监测终端 (温湿度/光照/烟雾/CO2/距离/语音控制) |

> 更多区域终端程序持续添加中...

## 技术栈

- **芯片平台**：海思 Hi3861V100 (WS63)
- **操作系统**：OpenHarmony LiteOS
- **通信协议**：WiFi 802.11b/g/n + MQTT
- **云平台**：华为云 IoTDA
